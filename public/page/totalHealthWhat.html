개요

종합 건강 지수는 각 오염물질농도 위험수치에 가중치를 부여하고 내부, 외부 작업 환경과 같은 환경요인을 고려하여 개발된 대기오염도 표현 방식

지수 산출 방식

  • 6개 대기오염물질별로 가중치를 산정하여 모든 값을 더함
  • 산출된 점수에 작업시간 작업환경별 가중치를 곱함
  • 종합 건강 지수는 0에서 100까지 4단계로 나누어 점수가 커질수록 대기 상태가 좋지 않음을 나타냄
  • PM2.5 = 초미세먼지
  • O3 = 오존
  • NO2 = 이산화질소
  • SO2 = 이산화황
  • CO = 일산화탄소
  • PM10 = 미세먼지
  • WTI = 작업시간지수
  • EF = 환경요인
